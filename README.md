# JobTracks AI

JobTracks AI is a full-stack job application tracker that helps users manage job applications, monitor progress, and receive personalized job recommendations based on their application history and preferences.

## Features

- User authentication
- Job application tracking
- Application status management
- Dashboard overview
- AI-powered job recommendations (planned)
- Resume-to-job fit analysis (planned)

## Tech Stack

### Frontend
- Next.js
- TypeScript

### Backend
- Django
- Django REST Framework

### Database
- PostgreSQL

### Infrastructure
- GitHub Actions
- Docker (planned)
- AWS (planned)
